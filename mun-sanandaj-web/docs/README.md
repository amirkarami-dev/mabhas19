# mun-sanandaj-web — Developer Docs

Learning-oriented documentation for the Sanandaj Municipality web service.
Docs are grouped by topic; each doc states its **level** at the top so you know where to start.

## Topics

### 🔄 Overview (`docs/`)
| Doc | Level | What you learn |
|-----|-------|----------------|
| [Full Work Flow](full-workflow.html) | Beginner | The whole service end to end, in simple words: the two background robots, how they read the city database and send reports/maps, the add-engineer retry, and what the page shows. Open the `.html` in a browser. |

### 🔐 Authentication (`docs/auth/`)
| # | Doc | Level | What you learn |
|---|-----|-------|----------------|
| 01 | [How Login Works](auth/01-login-flow.md) | Beginner | The full OIDC login flow — from clicking "Enter" to seeing the Dashboard, plus roles, logout, and staying logged in. |
